## Cách deploy nhanh lên Vercel
1. Vào Vercel và đăng nhập.
2. Chọn **Add New Project**.
3. Kéo thả toàn bộ thư mục này hoặc import repo GitHub chứa các file này.
4. Vercel sẽ tự nhận đây là static site.
5. Nhấn **Deploy**.
6. Sau khi deploy xong, bạn sẽ nhận được một link dạng `ten-du-an.vercel.app`.

## Nếu muốn gắn domain riêng sau này
1. Mua domain ở nơi bạn muốn.
2. Trong Vercel, vào project này.
3. Chọn **Settings** → **Domains**.
4. Thêm domain của bạn.
5. Làm theo hướng dẫn DNS của Vercel.
